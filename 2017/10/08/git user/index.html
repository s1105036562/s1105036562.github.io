Git 记录账户信息问题

  • 默认 Git 会记录你第一次登陆的用户名密码凭据
  • 如果想要在自己的本机切换多个账户，要记得去 控制面板中的 Windows 的 凭据管理器 中 Window删除 Git 记录的那个账户信息 然后再push时 就会让你输入用户名和密码
  • 如果你不想让 Git 帮你记录账户信息，则重新安装 Git ，在安装向导的过程中，把一个选项去除 credential
